LOREM APPSUM
Boilerplate web app starter for Angular 1

To use:

1. For testing, generate new self-signed TLS/SSL private key and certificate

  $ openssl req -x509 -newkey rsa:2048 -keyout key.pem -out cert.pem -days 1024 -nodes

  Simply tab past any options

2. Add your server/env/development.js to your .gitignore
